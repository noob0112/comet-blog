# Internationalization (i18n)

Multi-language support via `@nuxtjs/i18n`:

- **Supported languages**: English (default), Vietnamese
- **Translation files**: `i18n/locales/en.json`, `i18n/locales/vi.json`
- **Strategy**: `prefix_except_default` (English no prefix, Vietnamese uses `/vi`)
- **Cookie persistence**: User language preference saved in `i18n_locale` cookie
- **Browser detection**: Auto-detects browser language on first visit

## i18n Usage

```vue
<!-- In templates -->
<template>
  <h1>{{ $t('home.greeting') }}</h1>
  <p>{{ $t('footer.copyright', { year: 2025 }) }}</p>
</template>

<!-- In script -->
<script setup>
import { useI18n } from 'vue-i18n';
const { t, locale } = useI18n();
const message = t('common.readMore');
</script>
```

Key component: `<LanguageSwitcher />` toggles between languages
